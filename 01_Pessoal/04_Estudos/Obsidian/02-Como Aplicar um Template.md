## Introdução

Aplicar um template deixa suas notas consistentes e rápidas de criar. O processo depende do plugin que você usa — **Templater** ou **Core Templates**. Aqui está o passo a passo para ambos.

### 🔧 Usando o Plugin _Templater_ (recomendado)

1. Crie uma nova nota (`Ctrl + N`).    
2. Abra o menu de comandos (`Ctrl + P`).    
3. Digite **Templater: Insert template**.    
4. Escolha o template desejado (ex.: _Nota Diária_, _Projeto_, _Reunião_).    
5. O conteúdo será inserido automaticamente na nota.
    

**Dica:** Você pode configurar _hotkeys_ para templates específicos, deixando ainda mais rápido.

### 📄 Usando o Plugin Nativo _Templates_ (Core Plugin)

1. Certifique-se de que o plugin **Templates** está ativado em _Settings → Core Plugins_.    
2. Crie uma nova nota.    
3. Pressione `Ctrl + P` para abrir o menu de comandos.    
4. Procure por **Insert template** ou **Modelo**.    
5. Escolha o template que deseja aplicar.
