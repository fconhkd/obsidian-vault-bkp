## Introdução

Essa nota é para explicar o funcionamento basico do Entity Framework em um projeto dotnet, vou por o básico para me lembrar por onde começar

## Pré-requisitos

#### Instalar a ferramenta de linha de comando

As ferramentas de CLI .NET requerem o SDK .NET, `dotnet ef` deve ser instalado como uma ferramenta global ou local.
```bash
dotnet tool install --global dotnet-ef
```
- Para atualizar as ferramentas, use o comando.`dotnet tool update`    
- Instale o pacote mais recente.`Microsoft.EntityFrameworkCore.Design`
```bash
dotnet add package Microsoft.EntityFrameworkCore.Design
```
